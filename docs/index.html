Honest limits

The site should make the current scope clear before anyone evaluates the project.

  • VibeBench Arena is not a replacement for SWE-bench.
  • It is not a hosted enterprise platform yet.
  • It does not guarantee stars, funding, adoption, or correctness.
  • It is a local-first quality-control layer for AI-assisted coding workflows.
