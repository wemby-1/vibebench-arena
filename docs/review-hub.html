Use these commands to see the public evidence flow without a hosted account.

Demo

python3 -m vibebench demo

2

python3 -m vibebench evidence-room --output-dir PATH --zip

CI plan

python3 -m vibebench ci --dry-run --json

python3 -m vibebench evidence-room --verify PATH

CI artifacts to inspect

Successful GitHub Actions runs publish downloadable evidence bundles.

vibebench-proof-packet

Proof HTML, Markdown, JSON, manifest, and archive for review.

vibebench-site-preview

Static public docs preview with site-check output.

vibebench-evidence-room

Combined evidence-room package with index.html, trust-center.html, review-scorecard.html, proof packet, and site preview.

Local reproduction commands

Use a writable directory for PATH.

python3 -m vibebench proof --output-dir PATH --zip
python3 -m vibebench site-preview --output-dir PATH --zip
python3 -m vibebench evidence-room --output-dir PATH --zip
python3 -m vibebench ci --dry-run --json

After generating an evidence room, open index.html first, then use trust-center.html for project-maintained boundaries and review-scorecard.html or review-scorecard.md as the neutral checklist. The Trust Center is not a third-party certification, and the scorecard is a reviewer aid, not a third-party endorsement.

Verifier commands

Verify the package type you downloaded or generated.

python3 -m vibebench proof --verify PATH
python3 -m vibebench site-preview --verify PATH
python3 -m vibebench evidence-room --verify PATH
