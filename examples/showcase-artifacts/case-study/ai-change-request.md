# AI Change Request

## Request

Use an AI coding agent to add a small feature and update the related documentation.

## Review Concern

The change may look plausible in the editor, but the reviewer still needs evidence before trusting it.

The reviewer wants to inspect:

- what changed
- which checks ran
- whether risk findings appeared
- which artifacts were produced
- whether release readiness checks surfaced concerns

## Expected Review Packet

The useful output is not only the agent's explanation. The useful output is a review packet containing local checks, risk scoring, artifact inventory, comparison summary, and release readiness evidence.
